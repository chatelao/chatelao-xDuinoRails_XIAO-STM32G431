# Instructions for Agents

## Workflow

1. **Before Every Task**: Read `ROADMAP.md` to understand the current project status, upcoming goals, and context.
2. **At the End of Every Task**: Update `ROADMAP.md` to reflect the progress made.
    - Mark completed items as done (e.g., change `[ ]` to `[x]`).
    - Add any new necessary steps or notes based on the work completed.
